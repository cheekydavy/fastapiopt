FAQ
Is this completely free?

Yes. No sign-up, no subscription, no hidden fees. Download as much as you want.

Do I need to install anything?

Nothing. It runs in your browser tab. Works on Chrome, Firefox, Safari, and Edge.

Why do YouTube downloads have a search option?

For convenience — you can type a song name and we find the YouTube link for you automatically, then you just hit download.

Why does TikTok video have no watermark?

We use the tikwm API which pulls the original source file before TikTok applies the watermark overlay.

How long do downloads take?

YouTube and X stream in real time — usually 5–30 seconds. TikTok, Instagram, and Facebook download to our server first then transfer to you, so slightly longer for large files.

Is downloading videos legal?

Only download content you have permission to use or content that is in the public domain. Always respect copyright law and each platform's terms of service.
